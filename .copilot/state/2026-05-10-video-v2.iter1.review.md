## Nits (follow-up ok)
1. `UploadService.cs` contains three types (`UploadService`, `ConcatStream`, `CappedHashingStream`); split helpers into a `Storage/Streams/` folder when V2.5 chunked upload lands.
2. `StreamController` `/blob` endpoint returns `NotFound` for "asset row missing" vs "file missing"; consider a single sanitised 404 to avoid leaking which condition failed.
3. The test-side `SignedUrlForTests` mirrors the production HMAC composition; extract a shared `internal` helper in V3 so the format can't drift.
4. `LocalFsObjectStore.ResolveStrict` rejects backslash anywhere — fine for our Unix-leaning forward-slash key convention, but worth a one-liner clarifying it's deliberate.
