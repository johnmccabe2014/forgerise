   additive — no schema churn under V2/V3.
2. Pure scaffolding → reviewable in one PR, low blast radius.
3. Lets us bottom-out the four open architecture questions before any
   user-facing work starts.

V1 ships with `IObjectStore` + `IVideoJobQueue` defined as interfaces only —
no concrete provider yet — so none of those answers gate this slice.
